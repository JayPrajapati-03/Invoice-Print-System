# Vista Prints - Invoice Management System

A sleek, modern, and highly functional invoice printing application designed for professional printing services. Built with Next.js 16, this application provides a premium user experience with a focus on visual excellence and ease of use.

## ‚ú® Features

- **Premium Invoice UI**: A beautiful, "glassmorphism" inspired invoice design with a professional layout.
- **Dynamic Data**: Invoices are populated with dynamic data (currently fetching descriptions from a mock API).
- **Multi-Copy Support**: Easily print 1, 2, or 3 copies of an invoice with a single click.
- **Print Optimized**: Automatic layout adjustments for standard A4 desktop printing, including cleanup of UI elements and clones.
- **Micro-Animations**: Smooth hover effects and transitions for a high-end feel.
- **Watermark System**: Includes a stylish "PAID" watermark for processed invoices.
